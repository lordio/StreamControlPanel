###########################################################
#                         SETUP                           #
###########################################################
	You will need the Microsoft Visual C++ 2013 
Redistributable Package in order to run this program. 
	If you need to install it, it can be downloaded from 
http://microsoft.com/en-us/download/details.aspx?id=40784
You'll need the vcredist_x86.exe file.

This program has been built for 32-bit systems, for maximum
	compatibility.

###########################################################
#                         USAGE                           #
###########################################################
	Copy the signup list from Challonge into players.txt.
It shouldn't matter if you do it before or after you start
the program, but if done after, you will need to click
"Apply Titles" to load them.

	The score boxes will automatically update when
changed. The text boxes and dropdowns will wait until the
"Apply Titles" button is pressed.
